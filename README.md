## Requirements
+ docker
+ docker-compose

## Usage

+ Build Image
```
docker compose build
```

+ Deploy
```
docker compose up -d
```

+ Play

```
ssh player@127.0.0.1
```

> Password: isc

+ Shut off

```
docker compose down
```
