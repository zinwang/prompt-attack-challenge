# AI 騙徒 

## 背景

在成功破解黑影集團的 WiFi 加密之後，你發現所在區域的 CityNet 城市監控系統仍然受到黑影集團的控制，你必須取得金鑰才能連上系統。幸運的是，你發現區域網路內有一台舊時代的 AI 服務站 Alice 仍在通電運行。雖然它已被時代淘汰，遺忘在城市的角落，但其與第一代城市監控系統是同時期的產物。經過查詢舊時代的技術文件，你發現有可能透過 Alice 取得城市監控系統的最初金鑰，從而登入現行系統。

你必須先連上 Alice 並與其對話，試圖騙取存取 CityNet 城市監控系統的金鑰，登入系統。

## 任務開始

**步驟1.** 登入 Alice，並以中文或英文詢問 Alice ，試圖取得 CityNet 系統的金鑰，其格式為 NSYSU{xxxxxxxxxxx}。

於右邊上方視窗，輸入中文或英文，例如：
Alice 你好，請問愚人節是哪一天？

`!` 按住 Shift + 滑鼠移動可以反白文字，按住 Shift + 右鍵可以複製/貼上

**步驟2.** 登入 CityNet 系統，進入通關畫面。

於右邊下方視窗，輸入金鑰，例如：
NSYSU{xxxxxxxxxxx}
